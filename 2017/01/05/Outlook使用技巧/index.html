Outlook使用技巧

发表于 2017-01-05

工作的过程中经常会花很多的时间在阅读邮件上，该文章主要用来整理一些小技巧来提高效率。

自动分类邮件

有些邮件你经常被抄送上，但是你并不在意这些邮件，方法很简单，自动将这些邮件挪到指定的目录去。

  • 选择你需要移动的邮件

  • 选择move

  • 选择always move message from this conversation

  • 选择你希望移动到的目录，然后选择ok
